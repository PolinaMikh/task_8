эта программа выводит список друзей человека по заданному id
id пользователя, список друзей которого хотим увидеть, хранится в файле id.txt
перед использованием необходимо:
- импортировать библиотеку vk_api
- перейти на https://vkhost.github.io/
- выбрать приложение 'vk.com', нажать на него, затем нажать "разрешить"
- скопировать часть адресной строки от access_token= до &expires_in в файл tgitocken.txt
